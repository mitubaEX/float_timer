Introduction                               *float_timer* *float_timer-introduction*

float_timer is timer. Show result as floating window.

==============================================================================
Usage                                                        *float_timer-usage*

Example:
>
    FloatTimer <minutes>
<

When run `FloatTimer 1`, program wait 1 minutes. After 1 minutes, show result
in floating window.
